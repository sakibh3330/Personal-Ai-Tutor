# 🧠 AI Code Tutor – Learn CS Without Copy-Pasting

An interactive desktop application that acts as a strict **programming mentor**, not a code generator. Built with **Python**, **Tkinter**, and **Google Gemini**, this tutor guides you through computer science problems by explaining concepts — never by writing solutions for you.

---

## 🎯 Why This Exists

Most AI coding tools generate answers instantly. That hurts real learning.  
This app follows a **teacher-first philosophy**:

- ❌ Never outputs solved code  
- ✅ Explains *why* your code is broken  
- ✅ Teaches concepts, logic, and debugging strategies  
- ✅ Only answers CS-related questions  

It’s built for students who want to **understand**, not just copy.

---


## 🚀 Features

- 🧭 **Domain-Restricted**: Strictly answers Computer Science, programming, and software engineering questions.
- 🚫 **Zero Code Solutions**: The AI refuses to write code — it teaches you to write it yourself.
- 🗣️ **Socratic Guidance**: Uses hints, analogies, and guiding questions to help you debug.
- 💬 **Clean Chat Interface**: Scrollable conversation window built with Tkinter.
- ⚡ **Powered by Gemini**: Fast, contextual responses from Google’s latest model.
